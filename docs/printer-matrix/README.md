# printer-matrix

プリンタごとの検証結果をここに蓄積します。

運用ルール:

- `template.md` を複製して `YYYY-MM-DD-<printer-model>.md` の名前で保存する
- `requested size` と `measured size` は mm 単位で記録する
- barcode scan に失敗した場合も、そのまま失敗内容を残す

最低限残す項目:

- printer model
- driver version
- label stock
- template version
- requested size
- measured size
- barcode scan result
